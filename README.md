# ULog Analiz

PX4 uçuş loglarını (.ulg) analiz etmek için web tabanlı araç.

## Kurulum

**Gereksinim:** Python 3.8+

```bash
# Windows
setup.bat

# Linux/Mac
chmod +x setup.sh && ./setup.sh
```

## Kullanım

```bash
# Windows
run_explorer.bat

# Linux/Mac
./run_explorer.sh
```

Tarayıcınızda otomatik açılacaktır.
